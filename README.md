# Full Stack Development Journey

This repository documents a hands-on learning path for becoming a full-stack developer. It starts with the fundamentals of building web pages and progressively adds interactivity and backend capabilities.

**Stack Overview**
- Frontend: Basic HTML, HTML5, CSS, JavaScript, React
- Backend: Node.js + Express

**Goals**
- Learn semantic markup and responsive layout with HTML & CSS
- Understand modern JavaScript (ES6+) and DOM manipulation
- Build component-based UIs with React
- Create RESTful APIs and servers with Node.js and Express
- Connect frontend and backend to build full-stack features
